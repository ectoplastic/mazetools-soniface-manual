# Project

## Mazes Deposit

Using the Tray, a Maze with the settings of all instruments can be quickly saved and reloaded.

It allows a Maze to be copied from one Pattern to another.

Tip: A faster and more sustainable alternative is to save and load a Preset.

## Load Maze

The current Maze will be replaced by the new Maze.

If only one Maze exists within the current Pattern, the settings will be overwritten.

Tapping twice activates loading.

## Add Maze

Tap twice to add the selected Maze to the Pattern. Afterwards, it will be displayed on the Mazes List.

## Delete Maze

Tap twice to delete the selected Maze from the tray. Please note that this step cannot be undone.

## Save Maze

Save the current Maze to the tray. Tap the save button twice to do this.

All settings of Gridsynth, Sampler and Rhythm Interface are saved.

A .Mazes file with the name of the session will be saved to the local folder.

## Reset Maze

Reset the current Maze to the default state by tapping twice.

This step cannot be undone.

## Sets: Save All Patterns

Save the entire composition of Patterns, Mazes, and settings.

Once a session is saved, the app saves a *.sets file to the local file directory (see FAQ).

A set file can be copied, pasted into the Soniface folder on another device, and opened (copying samples along if needed).

## Load Set

Tap twice to load the set. Please make sure that the previous set has been saved!

## Add Set

Add a Set to an existing set. A Set is added after the last Pattern of the current Set.

Make sure that the device has enough capacity of CPU, GPU and RAM to generally play a Set of more than 3 Patterns smoothly.

## Delete Set from list

Tap twice to delete the selected set from the list. Note, this step cannot be undone!

## Save Set

Save the current set to the set list of the current session.

A set can also be saved to another session. For this you have to change the session first.

When the session is loaded, you can save the set in the set list.

## New Set

Start from the beginning, delete the current set with all Patterns, Mazes and settings.

The saved content in the set list will not be deleted or changed.

## Projects: Save & Load Sessions

A session is a composition of Mazes and Patterns. Depending on the process, it can consist of several sets.

The current set of a session is shown at the end of the list.

When saving a new session, a new *.set and *.Mazes file with the name of the sessions is automatically created in the Soniface folder.

## Save Session

Saves the current Session as a new set in the Session's Set list.

## New Session

Tap twice to start from the beginning within the session. All Mazes and Patterns will be deleted.

The other saved contents of the session will not be touched or changed.

To save a new session, use the Save As function.

## Load Session

Tap twice to load a session. Afterwards the sets of the session can be loaded.

It is possible to save a loaded set into another session.

Load the session and save the set in the sets menu.

## Sessions Folder

Mazes can be collected in a virtual folder.

A *.Mazes file is created in the Soniface folder on your device.

This way you can take your Mazes with you into the full version and use them as a basis for compositions.

## Save New Session

Save a new Session. The current composition is automatically saved as a Set with the name of the Session.

With the new Session a *.Mazes and a *.Sets file is created in the local Soniface folder with the name of the Session.

The title can also be entered from the keyboard.

## Open Session

Tapping on the title loads a session. In the sets menu, all sets of the session are displayed.

Tip: The last set displayed is the most recent one.

## Maze Presets

Maze presets contain many variations of the Gridsynth. They also offer combinations of all instruments, chord progressions and visuals.

When loading a Maze Preset, the entire Maze is overwritten (no undo).

Maze Presets can be saved from the Gridsynth menu.

## Save Maze Preset

Save a custom preset and name it.

The preset will be saved to the local Soniface folder and can be shared this way (FAQ).

## Add Maze Preset

Load a preset into the current Maze.

Presets contain different settings of the Gridsynth.

By loading, the preset overwrites the current settings of all instruments of the Maze, if the preset brings its own settings.

## Projects Save & Load

Projects are saved as a session. Press Save as... to name and save a new session

This will automatically create a .Set and a .Mazes file in the local Soniface folder with the session name

A session can contain any number of sets, which are displayed in a list in the Sets menu.
