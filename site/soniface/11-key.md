# Key

## Legacy Tonal System

The old sound system had some inconsistencies. For new sessions, it is therefore disabled by default. In older sets, the old sound system is automatically enabled so that their original sound behavior is preserved. It can be switched on and off manually.

## Key: Tonality, Scales & Chords

Each Maze can have its own key, scale or chord selection. This affects the Gridsynth, Bass and Sampler. If a Maze has no notes selected in the Key Interface, it will take over the active notes of another Maze.

The ring around the 3 octave circles is a sequencer. Tapping the ring places the active notes in the sequence/ progression.

The Picker menu allows chords and scales to be selected for each key.

The notes of a module are displayed by the LivePad or the Keyboard.

## Key: Geometry of intervals

Tapping the notes activates the tonality for the Maze. Notice how geometric shapes like triangles (triads) or pentagons affect the tonality.

The ring around the octave circles is a sequencer. Tapping the ring places the activated notes in the sequence and plays them as a progression.

## Keyboard & MIDI

Notes and chords can be played via the keyboard of the device or via a MIDI keyboard.

Tip: If a sequence is active, the sound jumps back to the sequence after an input.

If no sequence is active and no Chord is active in any other Mazes, the note input refers to all instruments in all Mazes.

## Paste to Sequencer

Pastes the current selection of notes at the paste position into the sequencer.

## Set notes on this position

The ring around the 3 octaves is the Chord Sequencers.

The Paste Position dot indicates the position where the current notes should be pasted in the Sequencer.

This function is an old method for creating a sequence.

## Progression Shift

All notes of the sequence will be shifted according to the quantization.

## Color Sequencer

The 12 rings of the Sequencer represent the 12 available root notes. A multi-sound is created when more steps are added.

The length of the sequence can be divided or multiplied by the speed.

The notes of the instrument are always shown by the LivePad or the keyboard.

## Play Sequence

Stopping the sequence means holding a note, chord or scale.

When the sequence is played again, jumps back to the sequence and the chords and scales within it.

Tip: If no sequence is active and no notes are active in any other Mazes, the note input applies to all instruments in all Mazes.

## Sequence Speed

The sequence can be slower (x0.5)or faster (x2) in relation to the main tempo.

The sequence phase is thereby divided (left) or multiplied (right).

The normal state is in the middle position.

## Clear Sequence

Delete all steps of the sequence/ progression.

## Rotate Chord

Slide the slider to the left or right to rotate the chord and change the root note (the lowest note).

## Root, Chord & Scale

In the Chord Picker menu, chords and scales can be selected for each key.

The ring around the sliders is the Chord sequencer. Tapping the ring places the active notes in the sequence.

## Root Note

## Scale selector

## Progression Step

## Color Editor

Here you can manually color a Maze and thus the environment on the Grid.

By default, the color of a Maze is determined by its pitch in the Keys menu or it receives a subtle automatic color. In this menu, Mazes can now be colored manually according to hue, saturation, and brightness.

## Color Editor Hue

## Color Editor Saturation

## Color Editor Value

Changes the brightness of the selected color of a Maze.
